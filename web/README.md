# Report Viewer

This folder contains a simple web UI that displays the JSON report generated by `code-analyses/index.js`.

## Prerequisites

- Node.js 18+ (or compatible)
- A report generated by running the analyzer (see below)

## Install

From the repo root:

```bash
npm install
```

## Generate the report

From the repo root:

```bash
npm run analyze
```

This creates/updates `reports/report.json`.

## Start the viewer server

From the repo root:

```bash
npm start
```

Then open:

- `http://localhost:3000/`

## How it works

- The UI is served from `web/index.html`.
- The UI fetches the report via the API endpoint: `/api/report`.
- The report file is expected at `reports/report.json`.

## Troubleshooting

- If you modify the scanner or report format, the UI may need updates.
- If the browser shows "Unable to load report", make sure `reports/report.json` exists and the server is running.
- If you run the server on another port, open the matching URL in your browser.
